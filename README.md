# KillerBee

### Installation

```bash
git clone 
cd KillerBee
```

### Usage
To run the database, run the following command:
```bash
docker-compose up
```

To run the api, run the following command:
```bash
cd backend
npm install
npm run dev
```

To run the frontend, run the following command:
```bash
cd frontend
npm install
npm run dev
```
